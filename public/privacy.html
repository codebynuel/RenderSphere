Privacy Notice

This MVP collects only what is needed to create accounts, submit render jobs, and deliver results.

Data We Store

RenderSphere stores account email addresses, hashed passwords, hashed API keys, session records, upload metadata, job metadata, uploaded `.blend` files, and completed render outputs.

How Files Are Used

Uploaded files are sent to object storage and opened by a RunPod worker to produce the requested render. Files are not used to train models or sold to third parties.

Retention

During the MVP, retention may be manual while lifecycle rules are finalized. If you need files or account data removed, contact support.

Security

Passwords and API keys are hashed before storage. Render files are still sensitive project assets, so avoid uploading confidential client work until the service has completed a fuller security review.

Contact

Privacy questions or deletion requests: support@rendersphere.app.
